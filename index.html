albedo

you can get a discount by solving the sequence

what to do

  • send a photo of your receipt to instagram and write the albedo you want
  • you'll get your sequence and figure out the formula and the next number in it
  • you can stretch out time and think as much as needed
  • if the answer is correct, you'll receive a 1-time promo code

rules

  • each subsequent number is obtained using the previous number only
  • the result may not be an integer, but it should be rounded and used as one
  • the maximum number of times a number can be used is 7

here, the list of possible operations is: +, -, ·, /, % and duplication (eg. 3 → 33 or 3,3)
